What is Vue?

"Vue is a progressive framework for building user interfaces ... designed from the ground up to be incrementally adoptable."

"The core library is focused on the view layer only, and is easy to pick up and integrate with other libraries or existing projects."
Features
  • Low barrier to entry
  • Resusable components
  • Directives (built in like v-once, v-if, v-else, v-show, v-for, v-html, v-on, v-bind or custom directives)
  • Filters
  • Mixins
  • By default escapes html to prevent cross site scripting when using the {{}} string interpolation
  • Focus on the core library, with concerns such as routing and global state management handled by companion libraries
More features
  • One way data flow by default lowers maintenance
  • Babel plugin allows the use of JSX if you really want
  • Virtual DOM, only updates the real DOM if a change actually changes something
  • DOM update event queue, attempts to queue up updates to the DOM
  • Server side rendering
  • Animations
  • OOOOODLES and BOOODLES more!!!!!1111!!!
